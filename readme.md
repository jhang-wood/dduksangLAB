# 떡상연구소

AI 기반 교육 & 커뮤니티 플랫폼. 강의 시청, 유저 커뮤니티, SaaS 홍보 기능을 중심으로 구성된 수익형 웹서비스.

## 🎯 목적
- AI/노코드 교육 콘텐츠 제공
- 수강생 간 커뮤니티 활성화
- 개인/팀 SaaS 홍보 플랫폼 구축

## 🛠️ 기술 스택
- **프론트엔드**: Next.js (App Router), Tailwind CSS, shadcn/ui
- **백엔드/DB**: Supabase (Auth + DB)
- **결제**: PayApp (추후 Toss 준비)
- **배포/인프라**: Vercel (예정)

## 📦 주요 기능
- 동영상 강의 시청 기능 (플레이어 + 이력 추적)
- 방문자 → 구매자 분류 로직
- 관리자 전용 페이지
- 커뮤니티 게시판 기능
- SaaS 카드형 홍보 영역 (급상승/신규 등)
